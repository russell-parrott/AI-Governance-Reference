Control Systems

An automated system that runs without human attention is a system without meaningful oversight. Effective governance requires that humans can see what the system is doing, respond when it behaves unexpectedly, intervene when necessary and stop it when required. Each of those capabilities depends on specific mechanisms being in place, mechanisms that must exist before they are needed, not be improvised in response to a problem.

Control Systems explains the operational concepts that make human control real: monitoring, alert systems, escalation procedures, human review, intervention authority, change management and system shutdown. Together these concepts describe the difference between a system that is governed and one that merely operates. When regulators investigate automated decision failures, the presence or absence of these controls is among the first things they examine.
